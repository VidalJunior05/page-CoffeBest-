## 📒🧾 About The Project
<br />

<ul>
    <li> Project developed using css FleBox and css Grid</li>

    <li> Custom ScrollBar</li>
    <li> Arrow with option to return to top of site</li>

```bash 
    #clone the repository

    # Enter directory
    $ cd page-CoffeBest

    # install dependencies
    $ yarn install

    # Initialize project
    $ yarn start
```
